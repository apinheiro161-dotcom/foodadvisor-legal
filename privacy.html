Politique de Confidentialité — FoodAdvisor

Dernière mise à jour : 15 mai 2026

1. Présentation

FoodAdvisor est une application mobile développée par Alexandre Pinheiro, aidant les expatriés à trouver des équivalents alimentaires entre leur pays d'origine et leur pays de résidence.

2. Données collectées

FoodAdvisor collecte les données suivantes :

  • Adresse email — utilisée pour la création et l'authentification de votre compte
  • Localisation GPS — utilisée uniquement pour afficher les supermarchés proches de vous, jamais stockée sur nos serveurs
  • Préférences utilisateur — pays d'origine, pays actuel, langue, favoris et liste de courses, stockées localement sur votre appareil

3. Utilisation des données

Vos données sont utilisées exclusivement pour vous authentifier, afficher les magasins proches, personnaliser les équivalents alimentaires et sauvegarder vos favoris. Vos données ne sont jamais vendues ni partagées avec des tiers à des fins commerciales.

4. Stockage et sécurité

Les données de compte sont stockées de manière sécurisée via Supabase, hébergé en Europe. La localisation n'est jamais stockée sur nos serveurs. Toutes les communications sont chiffrées via HTTPS.

5. Services tiers

  • Supabase — authentification et base de données
  • Kassal API — données de prix des produits norvégiens
  • Expo — infrastructure de l'application mobile

6. Vos droits (RGPD)

Vous disposez des droits d'accès, rectification, effacement, opposition et portabilité de vos données. Pour exercer ces droits : apinheiro161@gmail.com

7. Suppression du compte

Vous pouvez supprimer votre compte directement depuis l'application dans Paramètres → Supprimer mon compte. Toutes vos données sont effacées dans un délai de 30 jours.
